# BetterReads — Brand Language & Feature Naming

This document defines the vocabulary of BetterReads. Every feature, status, and action
has a name that feels warm, literary, and distinctly ours. Not borrowed, not generic.

---

## Brand voice

**Tone:** Warm, encouraging, bookish without being pretentious. Like a friend who
reads a lot and genuinely wants to talk about it.

**Not:** Gamified, corporate, clinical, or tech-bro.

**We say:** "You've finished 12 books this year." ✅
**Not:** "You've logged 12 reading sessions." ❌

**We say:** "What are you in the mood for?" ✅
**Not:** "Filter by mood parameter." ❌

## 2. Book Metadata & Tagging

### Moods → **"Vibes"**

This is the most important rename. "Moods" is bland. "Vibes" is how readers actually talk
("I need something cosy", "I want something dark"). It's the heart of what makes
BetterReads different from every other tracker.

**The BetterReads Vibes vocabulary (multi-select after reading):**

| Vibe | Icon |

| Emotional 💙 | |

| Funny 😂 | |
| Hopeful 🌱 | |

| Reflective 🪞 | |

| Whimsical ✨ | |
| Inspiring 🔥 | |

| Uplifting ☀️ | |

"Cosy", "Romantic", and "Gripping" are ours — StoryGraph doesn't have these.
Each book page shows a **Vibe Cloud** (not a pie chart) where community-voted vibes
appear in different sizes based on how many people tagged them.

### Pacing → **"Tempo"**

| StoryGraph name | BetterReads name |
|----------------|-----------------|
| Slow | **Slow Burn** 🕯️ |
| Medium | **Steady** 📚 |
| Fast | **Page-Turner** 🚀 |

These three names mean something. "Fast" is vague. "Page-Turner" is what readers actually
say. Shown as a horizontal bar on the book page.

### Character tags → **"Who's in it"**

Shown as community-voted quick-tags on each book page:
- Character-driven · Plot-driven
- Loveable cast · Flawed protagonist · Diverse voices · Strong development

### Themes & tropes → **"What it's about"**

Freeform + curated tags. Shown as a tappable tag cloud on the book page.
Examples: slow burn romance / found family / unreliable narrator / dark academia /
enemies to lovers / grief / coming of age / heist / sapphic / neurodivergent MC

### Content warnings → **"Heads Up"**

Not "Trigger Warnings" (clinical) and not hidden away. "Heads Up" is warm and practical.
Three tiers: **A lot** / **Some** / **Briefly mentioned**
Community-sourced + author-confirmed notes.
Users set their personal Heads Up flags in Settings → "My Reading Comfort Zone" and
books show a small 🔔 badge if they overlap.

## 3. Progress & Journalling

| StoryGraph name | BetterReads name | Why |
|----------------|-----------------|-----|
| Progress update | **Check-in** | Casual, low-pressure |
| Reading journal note | **Margin Note** | The most bookish possible name. People write in margins. |
| Reading streak | **Reading Streak** ✅ | Universal, keep it |
| Streak dashboard widget | **On a Roll** 🔥 | Shows current streak with warmth |

**Margin Notes:** Private rich-text notes attached to a check-in at a specific page.
Can include quotes, reactions, thoughts. Aggregated in a **Reading Log** view:
a personal, chronological journal of everything you've written while reading.
Beautiful, intimate, something no other tracker does well visually.

**The Made for You page** has named shelves (shown as horizontal scroll rows):

- *"What your friends are reading"* — social signal

## 6. Social Features

| StoryGraph name | BetterReads name | Why |
|----------------|-----------------|-----|
| Buddy reads | **Reading Together** | Exactly what it is, no jargon |
| Readalongs (public) | **Open Reads** | Accessible, open-door feel |
| Book clubs | **Reading Circles** | Warmer than "club", more intimate |
| Page-locked reactions | **Page Notes** | Ties back to Margin Notes language |
| Activity feed | **What's Happening** | Simple |
| Friend's profile | *their display name + "is reading..."* | Personalised, narrative |

### Reading Together (Buddy Reads)
- 2–15 people, one book
- Leave **Page Notes** at specific pages — hidden from others until they reach that page
- Optional end date
- Show progress bars for each participant

### Open Reads (Public Readalongs)
- Anyone can join, up to 1,000 participants
- Same Page Notes system — fully spoiler-safe
- Browse by vibe, tempo, or genre
- Host sets a schedule with optional milestone pages ("meet at page 150 by Friday")

### Reading Circles (Book Clubs)
- Membership + meeting schedule
- **"What are we reading?"** — voting poll for the next pick
- **"The Discussion Board"** — message board for each book
- **"Meeting Notes"** — post-meeting summary anyone can edit
- **"Question Pack"** — auto-generated discussion questions for the current book (AI)
- Progress tracking per member vs. next meeting date

## 9. Notifications & Reminders

Notifications should feel like a friend nudging you, not an app nagging you.

## 13. Things we do differently to StoryGraph

1. **Vibes Cloud** instead of a pie chart — visual, tactile, more beautiful
2. **"Cosy"** as a vibe — surprisingly missing from StoryGraph, huge for the audience
3. **Margin Notes** — the reading journal, built as a beautiful personal reading diary
4. **Discussion questions for Reading Circles** — auto-generated by AI, saves time
5. **"What's Next?"** — framed as a question (inviting) not a command ("Pick for me")
6. **Heads Up** framing — makes content warnings feel supportive rather than clinical
7. **Reading Quests** — lighter and more personal than "challenges"
